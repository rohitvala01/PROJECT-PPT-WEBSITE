Chapter 1

Project Introduction

The NewsRovalt System is a web-based news application that provides users with real-time news updates from multiple categories such as technology, sports, business, entertainment, and health. It serves as a centralized platform where users can easily browse and access the latest news articles using external News APIs.

Centralized Platform

All news categories accessible from a single interface

Real-Time Updates

Instant news fetching via API integration

Responsive Design

Smooth experience across all devices

Problem Analysis

Existing vs New System

Existing System

  • Lack of centralization
  • Time-consuming browsing

NewsRovalt System

  • Centralized news platform
  • Faster access to information
  • Improved user experience

  • Category-based browsing

Chapter 7

System Testing

White-Box Testing

Examines internal logic, exercises all independent paths, tests loops at boundaries, and validates internal data structures.

Black-Box Testing

Focuses on functional requirements, checks for incorrect functions, interface errors, and performance issues.

Unit & Integration Testing

Verifies individual modules (auth, search, categories) and their interactions, ensuring seamless data flow.

Alpha & Beta Testing

Internal testing by team and faculty, followed by external testing by volunteers for real-world feedback.
